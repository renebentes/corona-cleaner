# Corona-Cleaner

> Extensão do Chrome para a cura do Coronavírus na Internet.

## Objetivo

Remover as menções ao Coronavírus do conteúdo das páginas na Internet.

## Features
* Localizar as menções ao Coronavírus
* Aplicar a vacina
